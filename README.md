# Minecraft Clone

A simple Minecraft clone built with Three.js.

https://minecraft-gemini3.pages.dev/

## Features

- Voxel-based world generation
- First-person controls (WASD + Mouse)
- Block placement and removal (Right click / Left click)
- Jumping and collision detection
- Shadows


## How to Run

1. run `python3 -m http.server` in the root directory
2. open http://localhost:8000 in your browser
2. click to lock the pointer and start playing.
